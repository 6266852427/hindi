वेबसाइट की भाषाएं

HTML (हाइपर टेक्स्ट मार्कअप लैंग्वेज)

HTML वेबसाइट के निर्माण की पहली भाषा है। ब्राउज़र में HTML का उपयोग सामग्री (जैसे पाठ, चित्र, वीडियो आदि) को लोड करने के लिए किया जाता है, HTML सीखना आसान है। वर्तमान में उपयोग किया जाने वाला HTML संस्करण HTML5 है..........

और अधिक जानें

CSS (कैस्केड स्टाइल शीट)

वेबसाइट निर्माण में CSS दूसरी भाषा है। वेबसाइट पेज को डिजाइन करने के लिए CSS का उपयोग किया जाता है क्योंकि HTML केवल सामग्री को लोड करता है लेकिन वेब पेज पर किसी विशेष स्थिति में सामग्री सेट करने के लिए हम CSS का उपयोग करते हैं..........

और अधिक जानें

JavaScript (स्क्रिप्टिंग भाषा)

वेबसाइट के निर्माण में तीसरी सबसे महत्वपूर्ण भाषा JavaScript है। JavaScript का उपयोग केवल वेब पेज के सामने के छोर के लिए किया जाता है। चीजों को सामने की ओर से गतिशील बनाने के लिये JavaScript का उपयोग किया जाता हैं..........
